人生就是要不断开始

新时代

随着短视频时代的来临，文字好像逐渐失去力量。在搭建博客的时候，输入法不断提醒我是不是要打”播客”。好像大家都已经失去了阅读的耐性。
一个超过30秒的短视频，好像大家都觉得很长很长。我们好像慢慢对文字失去了耐心。

最简单的方式才能持久

尝试过很多博客方案，自己买 vm 也试过。续费，迁移，文章一遍一遍不断丢失。感觉好像还是最简单的方式，免费，markdown 才能持久。

又一个新的开始

好像人就是不要开始新的东西，然后也会不断新的开始。

这次就来尝试通过 github 来做博客吧。
